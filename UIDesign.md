UI 原型结构设计

2、首页 / 总览页面（评委第一眼）

（1）	页面目标

快速说明：我们能监测什么、AI 做了什么、结果如何

（2）	页面结构

[ 左 ] 功能菜单
[ 中 ] 当前研究区地图 + 土壤质量综合指数热力图
 
[ 右 ] 综合指数评分 + AI 报告

关键 UI 元素

土壤质量综合指数（大号数值 / 仪表盘）
AI 自动解读示例：
“该区域整体土壤质量中等偏好，西北部存在轻度盐渍化风险。”

四、土壤指标反演页面（核心展示页）

1、页面目标

明确体现“AI 在反演土壤指标”

2、页面结构

[ 左 ] 指标选择 + 参数设置
-	有机质
-	含水量
-	盐渍化

[ 中 ] 反演结果空间分布图（分级色斑）

[ 右 ] 统计信息
-	均值 / 最大值
-	等级占比柱状图


反演过程有加载动画（体现 AI 正在计算）分级颜色直观（适合答辩）
五、AI 多模态分析页面（创新点）

1、页面目标
 
让评委明确：这是 AI，不是简单算指数

2、页面结构

[ 左 ] 多模态输入说明
-	高光谱
-	地形 / 气象

[ 中 ] AI 分析按钮 + 风险区高亮地图

[ 右 ] AI 智能结论文本框

3、示例 AI 输出

“综合高光谱与地形因子分析，该区域东南部存在潜在土壤退化趋势，建议重点监测。”
